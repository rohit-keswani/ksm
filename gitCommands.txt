Add remote :
  Origin : (Your forked Repo)
          adding origin URL : git remote add origin <url>
          removing origin : git remote remove origin

  Upstream : (URL of Forked repo)
          adding upstream URL : git remote add upstream <url>
          remove upstream URL : git remote remove upstream


  Check Status of files to be committed:

          git status

  Add files to be committed:

          git add .   #the dot denotes that you are adding all the files

  Commit Changes :

                  git commit -m "<message>"

git pull origin branch - to fetch to system from repo
git push origin branch - to push from system to github in my repo without changing master repo
